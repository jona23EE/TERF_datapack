# Troty Energy Research Facility (T.E.R.F)
### by jona23

A vanilla Minecraft technology datapack adding various multiblock machines
that enable mass-material manufacturing and processing — all while keeping
multiple reactors from exploding, and monopolizing off their power.

**Java Edition only. Play in English (US) or custom item names may not apply.**

## About
T.E.R.F is a fictional research facility refurbished inside an old facility
in a volcano, built for the purpose of studying fusion technology.
The datapack features custom tools, recipes, advancements, textures,
multiblock structures, and much more.

Configuration settings: `data/terf/function/config`
Per-player config: `/trigger terf_trigger`

Bug reports: Create an issue on the github or [make one on Discord.](https://discord.gg/cXExM7pN3g)
